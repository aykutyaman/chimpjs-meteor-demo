# chimpjs-meteor-demo
## installation

* Install Chimp globally using

```
npm install -g chimp
```

* Clone this repo and start the demo app

```
git clone git@github.com:aykutyaman/chimpjs-meteor-demo.git
cd chimpjs-meteor-demo
meteor
```

* launch Chimp

```
chimp --ddp=http://localhost:3000 --watch
```
